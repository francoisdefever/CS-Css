Pour créer un site "responsive" et "mobile first"

La théorie Media Queries

La spécification CSS3 Media Queries définit les techniques pour l'application de feuilles de styles en fonction des périphériques de consultation utilisés pour du HTML. On nomme également cette pratique Responsive Web Design, pour dénoter qu'il s'agit d'adapter dynamiquement le design à l'aide de CSS.
Pour les bonnes pratiques il est conseillé de créer une feuille de style pour chaque supports. Par exemple : style.css, styletab.css et stylesmart.css.

L'application Media Queries

Afin de linker le Html, les trois feuilles de style et appliquer différents attributs selon les tailles. Il convient d'utiliser dans le header par exemple :
<link rel="stylesheet" href="style.css">
<link rel="stylesheet" href="styletab.css" media="screen and (max-width:768px)">
<link rel="stylesheet" href="stylesmart.css" media="screen and (max-width:320px)">

Les expressions

  • all : tous types de supports ou média
  • aural : navigateur à synthèse vocale
  • braille : plages braille
  • embossed : périphériques imprimantes braille
  • handheld : mobiles autres que smartphone
  • print : pour l'impression
  • screen : pour tous les écran
  • tty : terminaux de type telescripteur
  • tv : téléviseurs
